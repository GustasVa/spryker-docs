The _Non-splittable Products_ feature allows controlling if items bought in quantity greater than 1 are grouped in the cart or processed as separate items.


To learn more about the feature and to find out how end users use it, see [Non-splittable Products feature overview](/docs/scos/dev/features/{{page.version}}/non-splittable-products-feature-overview.html) for business users.
